Prints the absolute path of the current working directory to the 
standard output. The current working
directory is stored in the \Variable{PWD} variable.

\begin{lstlisting}[style=Groovybash, label={lst:example_pwd}]
cd "/some/place"
pwd
\end{lstlisting}
