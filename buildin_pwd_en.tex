\label{par:pwd}
\parparameter{\parcommand{pwd}}

Prints the absolute path of the current working directory. The current working
directory is available with the variable \parcommand{PWD},
see \refpar{par:variable_pwd}.

\subparagraph{Example:}

\begin{lstlisting}[style=Groovybash, label={lst:example_pwd}]
cd "/some/place"
pwd
\end{lstlisting}
