\label{par:user_home}
\parparameter{\parcommand{USER\_HOME}\\\parcommand{HOME}}

The home directory of the current user; the default argument for
the \parcommand{cd} (\refpar{par:cd}) built-in command.

\begin{lstlisting}[style=Groovybash, label={lst:example_user_home}]
cd HOME
echo HOME
echo USER_HOME
\end{lstlisting}
